This is a basic system that uses Twilio's Fax API to send faxes. To set it up for yourself or your own friend group, you'll need a developer account from Twilio, which is out of scope of this document.

Once you have a funded Twilio account, you'll need a phone number, and two codes from the Twilio Developer Console.

Twilio Phone Number

Buy a phone number from Twilio to send faxes from it. This also goes into the .env file of this project: 

# find these at https://www.twilio.com/console
# TWILIO_PHONE_NUMBER="+15551234567"
# TWILIO_ACCOUNT_SID=XXXXXXXXXXXXXXXXXXXXXXXXXXXX
# TWILIO_AUTH_TOKEN=XXXXXXXXXXXXXXXXXXXXXXXXXXXX

Account Credentials

You'll also need your "Account SID" and "Auth Token", both of which are also in the console. These are secrets, so be sure not to share them with anyone! The values in .env are only visible to the owner of the project, so you can safely paste them there: 

# find these at https://www.twilio.com/console
# TWILIO_PHONE_NUMBER="+15551234567"
# TWILIO_ACCOUNT_SID=XXXXXXXXXXXXXXXXXXXXXXXXXXXX
# TWILIO_AUTH_TOKEN=XXXXXXXXXXXXXXXXXXXXXXXXXXXX

(You might see options to generate and use API Keys instead, but we always need to provide the Account SID, and to validate requests from Twilio we need the Auth Token as well anyway.)

A Secret!

This app will send a real fax to a phone number and charge you money for doing so, and we don't want just anyone to be able to do that! Pick out a secret phrase or password you can use to make sure only you and your friends can send a fax from the app, and add it to .env 

# let's not let just anyone fax you
SECRET=XXXXXXXX
